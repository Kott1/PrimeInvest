# PrimeInvest Лендінг

Сучасний адаптивний односторінковий сайт для інвестиційної компанії з інтерактивними елементами, реалізованими за допомогою jQuery.

## Live Demo
[Відкрити робочу версію сайту в браузері](https://kott1.github.io/PrimeInvest/)

##  Стек технологій
* **HTML5** (семантична верстка, методологія BEM)
* **CSS3** (адаптивний дизайн під усі пристрої, Flexbox)
* **JavaScript / jQuery** (анімація акордеона, слайдер відгуків, багатокрокова форма з валідацією, таймер зворотного відліку)
